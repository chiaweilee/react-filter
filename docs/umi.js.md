Use in umi.js
-------------

## Install

```
$ npm install babel-plugin-transform-react-jsx-filter --save-dev
```

## Configure

.umirc.js

```JavaScript
export default {
  extraBabelPlugins: [ "transform-react-jsx-filter" ]
};
```

.eslintrc

```json
{
  "extends": [
    "react-jsx-filter"
  ]
}
```
