eslint-config-react-jsx-filter
------------------------------

## Install

```
npm install eslint-config-react-jsx-filter --save-dev
```

## Usage

### Via .eslintrc (Recommended)

.eslintrc

```json
{
  "extends": ["react-jsx-filter"]
}
```

### Via package.json

package.json

```
{
  "eslintConfig": {
    "extends": [ "react-jsx-filter" ]
  }
}
```
